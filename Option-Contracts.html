Volatility Tax Link
It is Difference between AM & GM of Asset's Return. If AM-Return is \( \big[ \mu \big] \) then, the GM-Return is \( \big[ \mu - 0.5 \sigma^2 \big] \). This \( \big[ 0.5 \sigma^2 \big] \) is the Volatility Tax.

Why this happens ??? Assume that Arithmatic & Geometric Returns are \( \big[ r_A, r_G \big] \). Therefore, \( \big[ P_0 e^{r_G} = P_0 \big(1 + r_{A}\big) \big] \Rightarrow \big[ r_G = \mathbb{LOG} \! \big(1 + r_A \big) \big] \). Now, \( \big[ \mathbb{LOG}x \big] \) is Concave Function. So, it increasingly penalizes -VE Arithmetic Returns the more -VE they are.

In this case, a Portfolio’s +25% Average Arithmetic Return into a 0 CAGR i.e. Geometric Average Return. Hence, leaves the Portfolio with 0 Profit i.e. Volatility Tax.

Implied Volatility Link
This is basically the Volatility Value, when is applied to some Model (e.g. BS Formula), the Model will give SAME Derivative's Price (e.g. Options Price) as the Market Price. IV is essentially an output from Black-Scholes Model when it is matched with the existing option price.

Local Volatility
This is generally applied in some Stochastic Process i.e. \( \big[dS =\! \mu_S dt + \sigma_S dW_t \big] \). In this case \( \big[ \sigma_S \big] \) can be thought of Local Volatility e.g. like Instantaneous volatility e.g. \( \left( \sigma_S^2 = 2 \left[ \frac{ \left(\frac{\delta c}{\delta t}\right) \:+ qc \:+ \left( r\:-q \right) K \left(\frac{\delta c}{\delta K}\right) }{ K^2 \left(\frac{\delta^2 c}{\delta t^2}\right) } \right]_{K=S, T=t} \right) \) Link.
In local Volatility there is NO Random Process for the Volatility, so there is NO CORR between Underlying & Volatility. BUT, Volatility Skew comes from such CORR. Therefore, to Price EXOTIC Options, Stochastic Volatility is BETTER Link.
LV \( \:\ne \) IV except in the trivial case where both are constant i.e. BS World Link. However from IV I can get LV Link.
