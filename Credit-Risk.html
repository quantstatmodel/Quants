Prob. that : Bond rated below Caa will survive UNTIL 2nd year is : \( 1 -\! 29.384 =\! 70.616 \% \).

Conditional Prob. Hazard Rate = HR that : Bond rated below Caa will default DURING 3rd year, with NO DEFAULT earlier, is : \( 9.298/70.616 =\! 13.16 \% \).
Assume : HR at time \( \big[ t \big] \) is \( \lambda \big( t \big) \). AND, Cumulative Prob. of Survival till time \( \big[ t \big] \) is \( V \big( t \big) \). So,

   \( \left[ \left[ \frac{V \big( t \: +dt \big) \:- V \big( t \big)}{V \big( t \big)} \right] = - \lambda \big( t \big) dt \right] \Rightarrow \left[ \left[ \frac{dV \big( t \big)}{V \big( t \big)} \right] = - \lambda \big( t \big) dt \right] \)
               \( \Rightarrow \left[ V \big( t \big) = e^{-\! \int_{0}^{t} \lambda \big( \tau \big) \,d \tau} \right] \Rightarrow \left[ Q \big( t \big) = 1 - e^{-\! \int_{0}^{t} \lambda \big( \tau \big) \,d \tau} = 1 - e^{-\! \big[ \overline {\lambda \big( t \big)} \big] t} \right] \)

Using Corporate Bond's Spread
This is Approx. formula. This formula's logic is : A Bond's CS will be equal to, Bond's Default's Value multiplied by the Default Prob. Hence, \( \big[ s \:= \big(1 - R \big) Q \big] \). Important obsn Link :
This is basically, 2-Factor formula i.e. CS and RR.
Different Bonds will have different Maturities. Therefore, We can have Default Prob's Term-Structure.
In above formula, We should have \( \big[ s \: \leq Q \big] \). However, in Real-life, this is NOT the case.
Here, the Assumption is : Corporate Bond's Value only depends on PD. This is FALSE As, it also depends on Liquidity.
2
Using Corporate Bond's Spread - A MORE EXACT Calculation
Consider a regular coupon paying Corporate Bond with Yield \( \big[y\%\big] \) & RFR \( \big[r\%\big] \) & Life \( \big[T\big] \) years. Also consider that, during the life-time, this Bond can Default at times \( \big[ T_1, T_2, ..., T_n \big] \) which, usually coincides with the Coupon paying dates.

If \( \big[ Q \big] \) is the Unconditional Prob. of Default Per Year for \( \big[T\big] \) years then, At time \( \big[ T_i \big] \) LGD is \( \big[ \big( B_{r,\: T_i} - R \big) Q \big] \). Here \( \big[ B_{r,\: T_i} \big] \) is the Bond's Value if NO Default at that Time i.e. When, Bond's Value is calculated using Forward RFR.

So, the PV of above is \( \big[ \big( B_{r,\: T_i} - R \big) Q \big] e^{-r T_i} \). Hence Overall PV is \( \big[ \sum_{i=1}^{n} \big( B_{r,\: T_i} - R \big) Q \big] e^{-r T_i} \).

So, \( \big[ \sum_{i=1}^{n} \big( B_{r,\: T_i} - R \big) Q \big] e^{-r T_i} = \big[ B_{r,\: T_0} - B_{y,\: T_0} \big] \). Use this, to calculate \( \big[ Q \big] \). So if \( \big[ B_{r,\: T_0} - B_{y,\: T_0} \big] \) is HIGH then, \( \big[Q\big] \) will be HIGH (this happened during 2007 Crisis).

So, with Bonds for varying \( \big[T\big] \), We can build the Term-Structure of \( \big[Q\big] \). This is Risk-Neutral PD Estimate As, We discounted using \( \big[r\big] \). This is LESS then the Real-World PD Estimate because of the Sys. Risk i.e. Default from one Bond will have Ripple effect to Default from Another Bond. Also Unlike Equity, the NonSys. Risk can NOT be diversified-away because, Bonds' Return is VERY Skewed i.e. \( \big[99\%\big] \) Prob. for \( \big[7\%\big] \) Return BUT, \( \big[1\%\big] \) Prob. for \( \big[-\!\!60\%\big] \) Return due to Default. So, for Diversification benefit, We need VERY LARGE number of Bonds. So, Bond Traders will earn Extra Return to bear this Risk.

Using Equity Price (Merton's Formula)
Assume that, a Company has ONLY 1 Bond outstanding at time \( \big[ T \big] \). It's Equity, Value, Bond are \( \big[ E_t, V_t, B_t \big] \).

So, We must have \( E_T = MAX \big[ V_T - B_T, \: 0 \big] \).
This is CALL Option Payoff. So, \( E_0 = V_0 N \big( d_1 \big) - B_T e^{-rT} N \big( d_2 \big) \). We know \( \big[ E_0 \big] \) from the Stock Market. So, We can calculate \( \big[ V_0 \big] \).

The Risk-Neutral PD is \( N \big( \!\!-\!\!d_2 \big) \). This requires estimates of \( \big[ V_0, \sigma_V \big] \).
                                 For \( \big[ \sigma_V \big] \) We have : \( \left[ \frac{dE_0}{E_0} =\! N \big( d_1 \big) \frac{dV_0}{V_0} \frac{V_0}{E_0} \right] \Rightarrow \left[ \sigma_E = \sigma_V N \big( d_1 \big) \frac{V_0}{E_0} \right] \) Link.
Adjust : Derivative's Value for Counterparty PD i.e. cVA
Assume, I have a Derivative that lasts till time \( \big[ T \big] \). It's Default can happen at time \( \big[ T_1, T_2, ..., T_n=T \big] \).
        If, there is NO Default then, this Derivative's Value is \( \big[ f_{T_i} \big] \).
        The, Risk-Neutral PD at time \( \big[ T_i \big] \) is \( \big[ q_{T_i} \big] \). This is INDEP. of the Derivative's Value.

Here, My Exposure at time \( \big[ T_i \big] \) is equal to, the Potential Loss : \( MAX \big[ f_{T_i}, 0 \big] \).

So, My Risk-Neutral LGD is \( \big[ \sum_{i=1}^{n} q_{T_i} \left( 1-R \right) E \big[ MAX \big[ f_{T_i}, 0 \big] \big] e^{-\!r_{T_i}} \big] \). This Value to be deducted from the Derivative's Value with NO Default. Hence the cVA.

It is Easy to understand that, if My Derivative's only Payoff is at time \( \big[ T \big] \) then, the above cVA will be \( \big[ f_{T_0} \sum_{i=1}^{n} q_{T_i} \left( 1-R \right) \big] \). So, the Derivative's Adjusted Value will be \( \big[ f^{*}_{T_0} = f_{T_0} - f_{T_0} \sum_{i=1}^{n} q_{T_i} \left( 1-R \right) \big] \).

One such Derivative is ZCB, So \( \big[ B^{*}_{T_0} = B_{T_0} - B_{T_0} \sum_{i=1}^{n} q_{T_i} \left( 1-R \right) \big] \). Hence We have \( \left[ \frac{f^{*}_{T_0}}{B^{*}_{T_0}} \right] =\! \left[ \frac{f_{T_0}}{B_{T_0}} \right] \).
This gives \( \big[ f^{*}_{T_0} = f_{T_0} e^{- \big( y^{*} - y \big) T } \big] \). Here, \( \big[ y^{*}, y \big] \) are the Yields for My Counterparty's ZCB and Risk-Free ZCB.

This shows that: Any European style Derivative Credit-Adjusted Value can be calculated by increasing the Discounting Rate appropriately.
